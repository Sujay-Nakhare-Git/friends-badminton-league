# Friends Badminton League Season 3

A Vite + React + TypeScript web app for running the Friends Badminton League Season 3 doubles tournament.

## Features

- Add and remove doubles teams
- Generate a round-robin match schedule
- Enter scores for completed matches
- View live standings with points and score difference
- Admin-only editing with viewer-only public access
- Save tournament data in the browser with local storage
- Ready for free hosting on platforms like Vercel, Netlify, or GitHub Pages

## Admin Access

- Username: `admin`
- Password: `FBLSeason3!`

Note: this is a client-side admin lock for a static site. It is suitable for basic gating, not strong security.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Hosting

This app builds to static files in `dist`, so it can be deployed on most free static hosting providers.
